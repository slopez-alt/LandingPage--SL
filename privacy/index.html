Política de Privacidad

Última actualización: 2026

SL Accounting Services PR respeta la privacidad de las personas que visitan este sitio web o se comunican con nosotros a través de formularios, email, teléfono o WhatsApp.

Información recopilada

La información recopilada puede incluir nombre, teléfono, correo electrónico, tipo de servicio solicitado, mensaje enviado y datos básicos relacionados con la solicitud de evaluación inicial.

Uso de la información

Esta información se utiliza únicamente para:

  • responder solicitudes de contacto;
  • coordinar consultas o evaluaciones iniciales;
  • dar seguimiento a servicios solicitados;
  • mejorar la comunicación y experiencia del usuario;
  • analizar el rendimiento general del sitio web y campañas de marketing.

Herramientas de análisis

Este sitio puede utilizar herramientas de medición como Google Analytics, Google Tag Manager u otras herramientas similares para entender el uso general del sitio web, páginas visitadas, fuentes de tráfico y acciones realizadas, como clicks en botones de contacto o WhatsApp.

Información sensible

No se debe enviar información sensible, como número de Seguro Social, EIN completo, documentos contributivos, estados bancarios, contraseñas o información financiera confidencial a través de formularios generales de contacto.

Terceros

La información enviada por los usuarios no se vende a terceros.

SL Accounting Services PR puede utilizar proveedores externos para operar formularios, analítica, hosting, email, WhatsApp u otros servicios tecnológicos necesarios para administrar la página y responder solicitudes.

Aceptación

Al utilizar este sitio web, el usuario acepta esta política de privacidad.
